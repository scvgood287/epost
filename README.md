# epost

- ADD : 기능 신규 개발
- MODIFY : 기존 기능 수정
- REFACTOR : 기존 기능 개선
- FIX : 버그 픽스
- REMOVE : 불필요한 로직 제거
- COMMENT : 코드 리뷰 반영 결과
- git issue 관련 내용 상세
    - Resolve: #이슈 넘버  (이슈를 해결함)
    - Ref: #이슈 넘버 (이슈 참고)
    - Relate to: #이슈 넘버 (관련된 이슈)
- 예시
ADD / 00기능을 구현하였습니다. 

00기능을 구현하였습니다. 
~~ 코드를 수정하였습니다.
